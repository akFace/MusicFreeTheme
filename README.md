# MusicFree 桌面主题包

### 简介

提供两款纯白主题色，纯白红色和粉色

### 使用方法

- 下载各个文件夹中的`*.mftheme`文件放到指定位置，在主题设置中导入即可

- [MusicFree 官方使用方法](https://github.com/maotoumao/MusicFreeThemePacks)。

# MusicFree 桌面主题包

### 简介

提供两款纯白主题色，纯白红色和粉色

### 使用方法
 
- [📦️下载主题包](https://github.com/akFace/MusicFreeTheme/releases)

- 将`*.mftheme`文件放到指定位置，在主题设置中导入即可

- 详情请看：[MusicFree 官方使用方法](https://github.com/maotoumao/MusicFreeThemePacks)。
